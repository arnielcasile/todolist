## Quick Start
- Clone or download this repo and place it into your server.
- `composer install `
- `cp .env.example .env `
- `php artisan key:generate`
- Create database and modify .env with your DB name and Pusher credentials. OR import the dump sql
- `php artisan migrate `
- `php artisan db:seed `
- `php artisan serve `
